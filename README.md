# Flask URL Shortener

A simple URL shortener web application built using Flask and SQLite.

## Features
- Shorten long URLs
- Redirect using short codes
- Track visit count
- Delete URLs
- Custom 404 page

## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS (Jinja2)
- Gunicorn
- Render
